# time
동아리원들과 실시간 채팅 서비스

![image](https://github.com/zoouniak/time/assets/88364328/236fc357-e90a-44c4-85c7-a36ad4053cc1)

 
## 서비스 설명

실시간으로 동아리 원들과 소통하고, 동아리의 공지사항을 확인할 수 있는 서비스를 만들고 있습니다.   

## 주요 개발 내용

- passport 모듈을 이용한 구글 로그인
- mongodb 데이터베이스 연동
- 클라이언트와 서버 간의 통신
- 공지사항 CRUD(create,read,update,delete)
